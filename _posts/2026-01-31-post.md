从 26 年 Q2 开始，Tesla 将停止销售 Model S，X 这两款高端车型，腾出来的工厂面积用于 Optimus 的生产。

Model S, X 一直是 Tesla 的高端车型，按照原先的“宏伟蓝图“这两款车的历史使命是给公司带来一些现金流，拉高品牌形象，从而让 Tesla 能够推出更好的大众车型，并且迅速提升市场占有率。Model S，X 分别是 2012，2015 年推出，凭借 Model S 3.1sec 零百加速，Model X 的鹰翼门，Tesla 奠定了电动车绝对领先的品牌形象。Model 3 以及 Model Y 分别于 2016 以及 2019 年亮相，截止 25 年年底，Model 3，Y 累积销售可能超过了 800w 台。？Model Y 连续 3 年是全球最畅销车型。目前 Tesla Model S，X 在销量以及营收的贡献基本可以忽略不计了（可能 3% 以内），整个计划完成的超出预期。

但是想要提升 Optimus 的产能一定需要停止 Model S，X 的工厂吗？我觉得这个决策的核心不在于工厂的面积，而是整个公司的战略转移。之前 Musk 搞 X.ai，一个数据中心直接就建在了临时用地上，长期审批留到后面再谈。可想工厂面积的扩容也不会是很难的事情。Tesla 未来的重点被放在了 FSD，Robotaxi（cybercab），Optimus 上，这三者主要围绕自动驾驶，具身智能展开，非常集中在物理 AI。停止 Model S，X 从战略上服务了未来的工作重点，减少了各种设计，维护的工作，让整个公司更加聚焦。

从 Tesla（包括电车以及 FSD），SpaceX，StarLink，这些 Musk 过去的辉煌战绩来看，他本人的工作模式非常终局驱动（用他本人的话就是第一性原理），并且集中在工程实现上。他首先立一个在当时来看非常夸张的目标，这个目标虽然很多人会说无法实现，但主要是工程上难度极高，而不会是科学上存在阻碍。比如说纯视觉 FSD，虽然这个目标至今仍然有人认为是不可行的，但主要论点就是当前的自驾系统可靠性还不够，需要激光雷达来兜底。Musk 通过人类的例子来论证，图像信息原理上是足够的，最大的问题是软件层面，坚持搞纯视觉方案。虽然 Tesla FSD 一开始的进展不及预期，常有撞死人的情况（L2 的责任在驾驶员），但是随着深度学习技术的发展以及 Tesla 海量数据的积累，FSD 做成功只是时间问题。引用别人的一句话“立一个非常远大的目标，在比较长的时间里快速迭代“基本上可以概括 Musk 的作风。

Optimus 目前的状态还不及预期，一方面机械结构的设计以及产能还没有达到大规模生产，大规模使用的门槛，另一方面，其软件层面的迭代也还没有展示出通用的操作能力。在很多的应用场景来看，Optimus 可能还是有遥操的辅助。原先我对于 Optimus 的进展是不乐观的，因为其工程挑战是在太高，但是看到了 Musk 能够快速调整公司产品线来聚焦研发，我觉得长远来看，Optimus 的进度大概率还是会超出预期的。

虽然 Musk 现在是全世界的焦点，但是这个也不是一般人能学的来的。首先就是抗风险能力低，比如 Tesla 和 SpaceX，历史上都有过差点破产倒闭的情况。或许在其他世界线上，Musk 是一个破产商人的概率更大。正式这种高风险，高收益的做法，叠加上一些好的运气，Musk 成为了当前的首富，说什么都是对的。其次，能够以 Founder Mode 持续在一线工作，本身就是一种天赋。大多数人其实没有那么强的学习欲，好胜心，抗打击能力。
